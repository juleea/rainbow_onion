First time here?

Rainbow Onion is an x86 assembly simulator designed to help you learn and better understand x86 assembly. Rainbow Onion features an in-browser assembly code editor and real-time visualizations of register and memory contents as code executes.

To begin, move your cursor over any of the help icons to learn about Rainbow Onion's features. Then use our first tutorial to familiarize yourself with our tool!
