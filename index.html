First time here?

Rainbow Onion is an x86 assembly simulator designed to help you learn and better understand x86 assembly. Rainbow Onion features an in-browser assembly code editor and real-time visualizations of register and memory contents as code executes.

To begin, move your cursor over any of the help icons to learn about Rainbow Onion's features. Then use our first tutorial to familiarize yourself with our tool!

About

Our Mission

Rainbow Onion is the product of three Stanford students' efforts to ease learning and understanding assembly. Learning assembly is one of the more challenging endeavors in computer science education, we designed this product with computer science students in mind.

Students we talked to found tools like gdb to be difficult and lacking for a first-timer learning assembly, so we built a tool that paralleled the IDEs many students are already familiar with. Our goal is to help students focus on learning the concepts instead of learning a new tool, and we hope you enjoy it!

Instruction Set

Rainbow Onion currently supports a subset of the x86 assembly instruction set.

Addressing Modes

The editor accepts four addressing modes:

  • Immediate: $value, e.g. $20 or $0x14 for the constant integer value 20
  • Normal: (register), e.g. (%ecx) to access the memory address stored in the ecx register.
  • Displacement: displacement(reg), e.g. 8(%ebp) to access the memory address stored at offset 8 from the address stored in the ebp register.
  • Indexed: displacement(base register, index register, scale), e.g. 8(%ecx, %eax, 4) to access the memory address stored at 8 offset from the address in ecx + 4 * the address in eax.
